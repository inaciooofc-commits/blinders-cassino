# Deploy rápido

## GitHub
1. Extraia o ZIP.
2. Suba todos os arquivos para o repositório.
3. Commit.

## Supabase
1. Abra Supabase > SQL Editor.
2. Cole `database/SQL_PARA_COLAR_NO_SUPABASE.sql`.
3. Rode.
4. Resultado esperado no fim:
   `BLINDERS_0_3_8_IRIS_CONTROL_SUBMENUS_OK`

## Cloudflare Pages
1. Create project.
2. Connect to GitHub.
3. Framework preset: None.
4. Build command: vazio.
5. Output directory: `/`.
6. Deploy.
